# AO3 Site Skin: *neos-catppuccin*

A variant of [ZerafinaCSS'](https://github.com/ZerafinaCSS/neos) neos site skin, with [Catppuccin](https://catppuccin.com/palette/) Latte (light) and Mocha (dark) variants. Maintains ZerafinaCSS' vision of "a slightly modernized, semi-flat look and enhances the user experience."

Visit the [AO3 work](https://archiveofourown.org/works/65849527) for instructions.
